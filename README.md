# My Reads

A book managing app that lets you search for and then add books to your bookshelf!

## How to run

- [ ] Clone it (<https://github.com/SyedAman/my-reads.git>)
- [ ] Install `node_modules`
- [ ] Run the dev server

```shell
$ git clone https://github.com/SyedAman/my-reads.git
$ cd my-reads
$ yarn && yarn run start
```
